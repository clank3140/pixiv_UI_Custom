# なにこれ
Pixivのメニューバーが消えてしんどいという噂を聞いたので作ったものです。
以下のメニューがPC版トップページに横並びで出ます。
- 作品管理
- フォロー
- フォロワー
- ブックマーク
- 閲覧履歴
- しおり
- 設定

なので画像の最大幅・最大高さをブラウザサイズ（？）にしました。
